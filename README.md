# Anime Profile V2 🚀

O **Anime Profile** é uma plataforma otaku gamificada que combina conteúdo editorial (notícias, artigos, reviews) com uma experiência social imersiva. Este repositório contém a recriação do site original, evoluindo de um monólito WordPress para uma aplicação moderna full-stack.

## 🎯 Visão do Produto
Transformar a leitura passiva de notícias em uma jornada de progressão, onde usuários colecionam figurinhas, sobem de nível, conquistam emblemas e participam de uma comunidade ativa.

## 🛠 Stack Técnica
- **Framework:** [Next.js 15+](https://nextjs.org/ ) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/ )
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/ )
- **Banco de Dados:** PostgreSQL (Prisma/Drizzle ORM)
- **Qualidade:** Vitest (Unitários) & Playwright (E2E)
- **CI/CD:** GitHub Actions
